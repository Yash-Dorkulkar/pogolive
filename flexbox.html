Flexbox

Flexbox (Flexible Box Layout) is a layout model that allows items within a container to be aligned and distributed dynamically. It's particularly useful for creating responsive designs.

Basic Flexbox Example

Item 1
Item 2
Item 3

.flexbox-container {
    display: flex;
    justify-content: space-between;
}

.flexbox-item {
    background-color: #BB86FC;
    padding: 10px;
    margin: 5px;
    color: white;
}

Grid

CSS Grid Layout allows us to create two-dimensional layouts by defining rows and columns. It offers more control over both vertical and horizontal alignments compared to Flexbox.

Basic Grid Example

Item 1
Item 2
Item 3
Item 4

.grid-container {
    display: grid;
    grid-template-columns: 
      repeat(2, 1fr);
    gap: 10px;
}

.grid-item {
    background-color: #3700B3;
    padding: 20px;
    color: white;
}

Responsive Layout

This section demonstrates a layout that adapts to various screen sizes using CSS Grid and Flexbox together.

Responsive Flexbox and Grid Example

Flexbox Item 1
Flexbox Item 2
Flexbox Item 3

.responsive-container {
    display: flex;
    flex-wrap: wrap;
    justify-content: space-between;
}

.responsive-item {
    background-color: #03DAC5;
    padding: 20px;
    margin: 10px;
    color: white;
    width: 30%;
}

@media (max-width: 768px) {
    .responsive-item {
        width: 100%;
    }
}
