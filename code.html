HTML

HTML stands for HyperText Markup Language. It is the standard language for creating web pages.

Basic Tags

Here are some basic HTML tags:

  • <h1> to <h6>: Heading tags, where <h1> is the largest and <h6> is the smallest.
  • <p>: Paragraph tag, used for text content.
  • <a>: Anchor tag, used for links.
  • <img>: Image tag, used to embed images.
  • <ul> and <ol>: Unordered and ordered list tags, used for lists.

Example

<p>This is a paragraph.</p>

This code creates a paragraph that says "This is a paragraph."

CSS

CSS stands for Cascading Style Sheets. It is used to style and layout web pages.

Common CSS Properties

Here are some common CSS properties:

  • color: Sets the text color.
  • background-color: Sets the background color of an element.
  • font-size: Sets the size of the text.
  • margin: Sets the space outside an element.
  • padding: Sets the space inside an element.

Example

h1 { color: blue; }

This code changes the color of all <h1> elements to blue.

JavaScript

JavaScript is a programming language that allows you to implement complex features on web pages.

Common JavaScript Functions

Here are some common JavaScript functions:

  • console.log(): Outputs a message to the web console.
  • alert(): Displays an alert box with a specified message.
  • document.getElementById(): Accesses an HTML element by its ID.
  • addEventListener(): Attaches an event handler to an element.

Example

console.log('Hello, World!');

This code prints "Hello, World!" to the console.
